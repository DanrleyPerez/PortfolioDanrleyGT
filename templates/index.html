Sobre:

Aplicação web criada com o intuito de análise de mercado para o Marketplace do Elo7, visando auxiliar os pequenos vendedores na escolha de parâmetros para o anúncio de seus produtos. Em termo para busca, escolha o nome para o produto ao qual tem interesse de anunciar, e ao lado o número de produtos para a análise. Ao iniciar, será feita uma varredura em todas as páginas, trazendo as principais informações e métricas.

Principais tecnologias utilizadas nesse projeto :


Python

Alta velocidade e simplicidade para análise de dados.


Django

Framework muito utilizado no desenvolvimento web em empresas como: Spotfy, Youtube, Reddit.


Bootstrap

Para algumas intereções e estilos da página
